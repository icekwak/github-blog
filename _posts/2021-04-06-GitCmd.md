# Git 저장소 생성
현재 directory 기준으로 Git 저장소 생성.
```bash
$ git init
```

# 파일 상태 확인
Commit할 상태가 되었으면 초록색으로 안했으면 빨간색으로 표시됨.
```bash
$ git status
```

# Commit할 수 있게 stage에 파일 추가
모든 파일 stage에 추가 후 파일 상태 확인.
```bash
$ git add .
$ git status
```

# Commit
"commit comment" 부분에 Commit 내용 쓰기.
```bash
$ git commit -m "commit comment"
```

# push
원격 저장소에 push.
```bash
$ git push <저장소> <브랜치>
```
ex)
```bash
$ git push origin master
```
