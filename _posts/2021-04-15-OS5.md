# 프로세스 간 통신
### 종류
- 프로세스 내부 데이터 통신 (멀티스레드)
- 프로세스 간 데이터 통신 (파일, 파이프로 통신)
- 네트워크를 이용한 데이터 통신 (소켓, RPC로 통신)

### 통신 방향에 따른 분류
- 양방향 통신
  - 동시에 가능
  - 소켓 통신이 대표적
- 반양방향 통신
  - 동시에는 불가능
  - 무전기가 대표적
- 단방향 통신
  - 한 쪽만 가능
  - 전역 변수와 파이프가 대표적

### 통신 구현 방식에 따른 분류
- 대기가 있는 통신
  - 동기화를 지원함
  - 자동으로 대기 상태에 머물러 있음
  - 파이프, 소켓
- 대기가 없는 통신
  - 동기화를 지원하지 않음
  - 데이터가 도착 여부를 직접 확인
  - 전역 변수, 파일
